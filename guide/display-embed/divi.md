# Divi 5 Module (Pro)

FluentPlayer Pro includes native **Divi 5 modules** for both the video player and playlist. Add them to any Divi row or section and configure every setting directly inside the Divi builder panel — no shortcode required.

::: info Pro feature
The Divi 5 module requires **FluentPlayer Pro**. The free version supports [Shortcode](/shortcode) and [Gutenberg Block](/block) embeds.
:::

::: warning Divi 5 only
These modules are built for **Divi 5**. They are not available in Divi 4 or the Divi Builder plugin for classic themes. If you are on Divi 4, use the [Shortcode](/shortcode) embed instead.
:::

## Adding the FluentPlayer Module

1. Open a page in the **Divi** visual builder.
2. Click the **+** icon to insert a new module inside any row or section.
3. Search for **FluentPlayer** in the module search.
4. Click the **FluentPlayer** module to insert it.

5. In the module settings panel, open the **Content** tab and select the **Media** item from the dropdown.
6. The player renders live in the Divi canvas.

[Screenshot: FluentPlayer module in the Divi builder showing the Content tab with media selector and live player preview]

7. Click **Save** to apply your changes.

## Adding the FluentPlayer Playlist Module

1. Insert a new module and search for **FluentPlayer Playlist**.
2. Click the **FluentPlayer Playlist** module to insert it.
3. In the **Content** tab, select the **Playlist** from the dropdown.
4. Choose a **Layout** (Standard or Grid) and configure display options.
5. Click **Save**.

[Screenshot: FluentPlayer Playlist module in Divi showing playlist selector and layout options]

## Available Settings

The Divi module exposes the same settings as the [Gutenberg Block](/block-editor):

| Panel | What you can configure |
|-------|----------------------|
| **General Settings** | Media source, poster image, preset selection |
| **Branding & Appearance** | Logo, colors, player skin overrides |
| **Interactive Layers** | Email capture, CTA, Hotspots, Fluent Forms, Ads |
| **Chapters** | Chapter markers and titles |
| **Subtitles** | Subtitle tracks |
| **Multiple Languages** | Audio language tracks |
| **Timed Content (Pro)** | Content blocks that appear below the player at timestamps |
| **Overlays (Pro)** | Text and button overlays |
| **Advanced** | Custom CSS, visibility rules |
