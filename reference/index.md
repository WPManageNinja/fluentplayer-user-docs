# Reference Overview

This section provides quick-lookup reference pages for Fluent Player's shortcodes, settings, and technical details. Use it when you know what you are looking for and need the exact syntax, option name, or configuration value.

## What is in this section

| Page | What it covers |
|------|---------------|
| [Feature Catalog](/reference/feature-catalog) | Complete checklist of features (free vs Pro), embed methods, presets, integrations |
| [Shortcodes Reference](/reference/shortcodes) | Every shortcode with its attributes, syntax, and examples |
| [Settings Reference](/reference/settings) | Every settings section and option available in the admin |
| [Plugin Overview (Technical)](/reference/plugin-overview) | Architecture, REST API, and internal details for developers |

## How reference pages differ from guide pages

- **Guide pages** (in the sidebar under Get Started, Videos & Media, etc.) explain *how to do something* step by step, with context and examples.
- **Reference pages** list *every option* in a structured, scannable format — like a dictionary for the plugin.

If you are learning Fluent Player for the first time, start with the [Guide](/guide/). If you need to look up a specific shortcode attribute or setting, use the reference pages below.
