# Settings Reference

This page is a quick-reference index of all Fluent Player settings sections. For detailed documentation on each section, follow the links below.

::: info
Settings are found at **Fluent Player → Settings** in the WordPress admin. Fluent Player Pro unlocks additional sections. The inner sidebar inside the Settings screen lists only the sections available for your active license.
:::

## Settings sections

| Section | What it covers | Detailed page |
|---------|----------------|---------------|
| **General** | Default preset, aspect ratio, auto resume playback, global custom CSS | [General Settings](/guide/settings/general) |
| **Branding** | Brand color, control bar color, logo image, logo link, logo position — with Live Interactive Preview | [Branding Settings](/guide/settings/branding) |
| **Integrations** | Email/lead capture providers: FluentCRM, Webhook, Mailchimp | [Integrations Settings](/guide/settings/integrations) |
| **Storage** | Video hosting: BunnyCDN Stream, BunnyCDN Storage, Mux | [Storage Settings](/guide/settings/storage) |
| **YouTube** | Privacy-enhanced mode, Subscribe button for YouTube sources | [YouTube Settings](/guide/settings/youtube) |
| **Analytics** | Fluent Player Analytics tab + Google Analytics tab | [Analytics Settings](/guide/settings/analytics) |
| **Export** | Download email capture data as CSV, JSON, or ODS | [Export](/guide/settings/export) |

## Settings priority

Many global settings can be overridden at more specific scopes:

```
per-block / per-media  →  preset  →  global setting
```

The most specific scope always wins. See [Settings Overview](/guide/settings/) for more on scoping.
